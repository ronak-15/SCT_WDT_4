# **SCT_WD_4**

**To-Do Web App**

A beautiful, feature-rich To-Do web app built with pure HTML, CSS & JavaScript.

No frameworks. No dependencies. Just open and use.

👨‍💻 **Built With**

HTML5 — semantic structure

CSS3 — variables, grid, flexbox, animations, backdrop-filter

JavaScript (ES6+) — no frameworks, no build tools

localStorage API — client-side data persistence
__________________________________________________________________________________________________________________________________________

✨ **Features**
Task Management

FeatureDescription

➕ Add Task : Type a task and press Enter or click Add✅ 

CompleteClick :  the circle checkbox to mark done/undone✏️

 EditClick : the pen icon to edit all task details in a modal🗑️ 
 
 DeleteClick : the trash icon — animates out smoothly🧹
 
 ClearDone : Remove all completed tasks at once from the sidebar
